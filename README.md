## 📱 アプリの機能とコンセプト

### コンセプト
**「ついダラダラしてしまう時間を、可視化して管理する」**
一般的なTodoリストは「何をやるか」しか管理できませんが、このアプリは**「どれくらいの時間でやるか（Time-boxing）」**をセットにすることで、学生特有の「課題の先延ばし」や「集中力の欠如」という課題を解決するために開発しました。

### 主な機能
1.  **タイマー付きタスク作成**: タスク作成時に「10分/25分/45分/60分」から目標時間を設定できます。
2.  **集中モード（シングルタスク）**: 再生ボタンを押すとタイマーが減り始めます。一度に一つのタスクしか動かせないように制御し、マルチタスクによる効率低下を防ぎます。
3.  **プログレスバー**: 残り時間が視覚的に減っていくバーを表示し、締め切り効果で集中力を高めます。
4.  **データの永続化**: ブラウザを閉じてもタスクが消えないよう、LocalStorageに自動保存されます。

## 🔥 特に挑戦・工夫したところ（オリジナリティ）

### 1. ニッチなニーズへの対応（ポモドーロ・テクニックの応用）
汎用的なTodoアプリではなく、**「勉強や課題に取り組む学生」**という特定のターゲット層のニーズ（集中したい・時間を区切りたい）に特化しました。
単純なチェックリストではなく、「実行中（Active）」の状態を持たせることで、ユーザーに「今、自分はこのタスクに取り組んでいるんだ」という強い意識付けを行います。

### 2. React Hooksを活用した動的なUI制御
`useState` でタスクの状態を管理するだけでなく、`useEffect` と `setInterval` を組み合わせて、リアルタイムに減少するタイマーを実装しました。
また、**「あるタスクのタイマーをスタートさせると、他のタスクのタイマーは自動停止する」**というロジックを組み込むことで、強制的にシングルタスク環境を作る工夫をしました。

### 3. UX（ユーザー体験）への配慮
* **直感的な操作**: 再生・停止ボタンアイコン（Lucide React）を使用し、説明なしでも使えるUIにしました。
* **緊急度の可視化**: 残り時間が1分を切ると、タイマーの表示色が変わり、ラストスパートを促すような視覚的フィードバックを実装しました。
