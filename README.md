## 📱 アプリの概要とコンセプト

### コンセプト
**「タスク管理を、RPGのように楽しく」**
従来のTodoリストは「やるべきこと」を管理するだけの事務的なもので、義務感によるストレスが溜まりがちです。
このアプリは、**「タスク消化＝経験値獲得（レベルアップ）」**というゲーミフィケーション要素と、入力されたテキストから時間やカテゴリを自動判別する「スマート解析（AI Logic）」を組み合わせることで、モチベーション維持と入力の手間削減を同時に実現しました。

### 💡 主な機能

#### 1. ✨ スマート入力解析 (Smart Parser)
わざわざプルダウンで時間を選んだり、カテゴリ設定をする必要はありません。自然言語風に入力するだけで解析します。
* **「資料作成 45min」** → 「45分のタイマー」「Workカテゴリ」に自動設定。
* **「英単語暗記 30m」** → 「30分のタイマー」「Studyカテゴリ」に自動設定。
* 入力内容に基づいて、タスクの**難易度**と獲得できる XP（経験値）が自動計算されます。

#### 2. 🎮 ゲーミフィケーション・システム
タスクを完了するとXP（経験値）が手に入り、レベルが上がります。
* **レベルバッジ**: 現在のステータスを可視化。
* **XPバー**: 次のレベルまでの進捗をプログレスバーで表示。
* 完了時の演出やレベルアップアラートにより、ドーパミンを刺激し、継続率を高めます。

#### 3. 🔋 エネルギー・キャパシティ管理
「1日にどれくらい働けるか（例: 8時間）」を設定し、残りのタスク量と照らし合わせて「今日のエネルギー残量」を可視化します。
* タスクを詰め込みすぎると「⚠️ キャパシティ超過」のアラートが表示されます。
* 無理な計画を防ぎ、燃え尽き症候群（バーンアウト）を予防します。

#### 4. 🧠 隙間時間AI提案
「次の予定まで10分しかない…」そんな時、ボタン一つで**15分以内に終わるタスク**をリストからピックアップして提案してくれます。

## 🎨 UI/UX のこだわり

### モダンで没入感のあるデザイン
* **ダークモード調のアクセント**: `linear-gradient` を多用し、近未来的でワクワクするようなUIを構築しました。
* **インタラクティブなアニメーション**: タスク完了時のスライド動作や、XPバーの増加アニメーションなど、操作に対するフィードバックをリッチに実装しています。
* **緊急度の可視化**: 残り時間が少なくなるとタイマーが赤く点滅（Pulse Animation）し、ラストスパートを促します。

## 🛠 使用技術

* **Frontend**: React, TypeScript, Vite
* **Styling**: CSS3 (Custom Properties, Animations)
* **Icons**: Lucide React
* **Persistence**: LocalStorage (リロードしてもデータ・レベル・XPは保持されます)

### 工夫した実装ポイント（コード解説）

1.  **正規表現による自然言語処理風ロジック**:
    `App.tsx` 内の `parseInput` 関数にて、正規表現を用いて入力テキストから「時間（数字+m/min）」や「文脈（mail/study/gym等）」を抽出し、オブジェクトとして構造化するロジックを自作しました。

2.  **カスタムフックを使わないステート管理**:
    `useState` と `useEffect` を組み合わせ、タイマーの減算処理とLocalStorageへの同期をシンプルかつ堅牢に実装しています。

3.  **動的なスタイル制御**:
    残り時間や現在のXPに応じて、CSSの `width` プロパティや `background-color` を動的に計算し、Reactの `style` 属性経由で滑らかに変化させています。
